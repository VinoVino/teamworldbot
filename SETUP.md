1. Create a virtual environment.
`python3 -m venv venv`

2. Activate your virtual environment.
`. venv/bin/activate`

3. Upgrade pip
`python3 -m pip install --upgrade pip`

4. Install Flask
`pip install Flask`

5. Tell Flask how to import the app
`export FLASK_APP=emojify.py`

6. Run the app
`flask run`
